Casserole

 

Welcome to Casserole!

Casserole is a tool developed and distributed by ElMiller Technology. It is used by Chomebooks to boot them into developer mode. 

 

Casserole uses the Odyssey Api and browser storage to open SysInternals. Casserole also uses SessionStorage and LocalStorage to inject code into the Boot Sector. 

As of now Casserole is in development and is free, however once Casserole releases fully it is planned to be paid.
